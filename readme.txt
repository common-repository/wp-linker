== Description ==

Plugins adds dynamically links across your website according to your configured internal linking campaign.

== Frequently Asked Questions ==

= How to install and use this WP linker plugin? =

After you Active this plugin, just go to Dashboard > WP Linker > and then configure your internal linking campaign for your personal use

= How do I report bugs and suggest new features? =

You can report bugs or request new features in dedicated plugin support section

= Will you include features to my request? =

Yes, Absolutely! We would suggest you send your feature request by creating an issue in dedicated plugin support section
